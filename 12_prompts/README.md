# Canonical Prompts

This directory will contain only approved prompts.

Planned assets:

- Diagnostic prompt
- Diagnostic report interpreter
- Qualification prompt
- Messaging calibration prompt
- Industry research prompt
- Implementation planning prompt
- Outcome review prompt
- Learning extraction prompt

Each prompt must include:

- purpose;
- inputs;
- output schema;
- constraints;
- evidence requirements;
- confidence reporting;
- human escalation conditions;
- version;
- owner;
- tests.
