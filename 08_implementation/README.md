# Implementation System

## Working engagement model

- Approximately three hours each week
- Fixed monthly retainer
- One improvement at a time
- Existing people and tools first
- No large transformation project by default

## Weekly cycle

1. Review what changed.
2. Select one improvement.
3. Implement.
4. Document.
5. Transfer knowledge.
6. Measure.
7. Update the backlog.

## Weekly proof of value

At least one of:

- one process simplified;
- one piece of critical knowledge captured;
- one repetitive task reduced;
- one dependency reduced;
- one recurring mistake prevented;
- one person spends less time compensating;
- one operational risk made visible and controlled.
