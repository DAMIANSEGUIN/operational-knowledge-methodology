# Field Guide

The Field Guide is organized around the movement of a real organization from first signal to measurable improvement.

Planned states:

1. Identify
2. Research
3. Calibrate message
4. Begin conversation
5. Offer diagnostic
6. Receive and interpret response
7. Qualify
8. Review findings
9. Select implementation path
10. Implement
11. Measure
12. Retain, refer, or close
13. Capture learning

Every state must define:

- entry conditions;
- inputs;
- agent actions;
- human decision points;
- outputs;
- quality gates;
- failure modes;
- next-state rules;
- metrics.
