# Repository Validation Checklist

- `PROJECT.md`, `CURRENT_STATE.md`, `README.md`, `CHANGELOG.md`, `DECISIONS.md`, and `GLOSSARY.md` exist.
- Each top-level module has a `README.md` or canonical owner.
- Generated artifacts are separate from canonical source.
- Methodology follows the dependency order recorded in `PROJECT.md`.
- Gap analysis exists before methodology and field guide revision.
- Human judgment boundaries are explicit.
- Canonical prompts live in `12_prompts/README.md`.
- Canonical templates live in `13_templates/`.
- Source material is recorded in `15_source_material/`.
- Markdown links resolve.
- No unresolved draft markers remain in canonical Markdown.
- `CURRENT_STATE.md` and `CHANGELOG.md` reflect the current milestone.
- Significant design decisions are recorded in `DECISIONS.md`.
