# Gap Analysis

## Objective

Determine what must change in the current Operations Support methodology before the Field Manual is rewritten.

## Required evidence

- Current canonical Field Manual
- Current working strategy
- Current diagnostic prompt
- Current qualification prompt
- Current messaging assets
- Current implementation workflow
- Current deliverable set

## Classification

Each reviewed section must be classified as:

- Retain
- Strengthen
- Reframe
- Merge
- Remove
- Defer

## Required output

Use `GAP_ANALYSIS_TEMPLATE.md`.

Do not rewrite source material during review.
