# Qualification System

## Purpose

Interpret diagnostic results and lead responses to determine fit, need, timing, trust, implementation feasibility, and the next best action.

## AI responsibility

Produce a decision package.

## Human responsibility

Approve or revise judgments involving fit, messaging, trust, timing, pricing, and commitments.

## Possible outcomes

- Continue value-first conversation
- Ask targeted questions
- Offer review meeting
- DIY recommendation
- Guided implementation
- Fractional Operations Support
- Refer elsewhere
- No action
